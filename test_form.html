1. Куда слать

Ссылку выдаёт бот на команду /newkey. Если эта страница открыта не с того же компьютера, где крутится сервис (например, вы выложили её на GitHub Pages) — адрес обязательно должен быть публичным и по HTTPS. http://localhost:8000/... отсюда не достучится: браузер не пустит внешнюю страницу к вашему компьютеру, а страница на HTTPS ещё и заблокирует запрос к обычному HTTP («mixed content»).

2. Формат отправки

Сервис одинаково понимает оба формата — переключайте, чтобы проверить оба пути.
